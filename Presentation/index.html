HTML

HTML is the standard markup language for creating Web pages.

The time needed to learn: from 2 weeks to 1 month

CSS

CSS is the language we use to style a Web page.

The time needed to learn: from 3 weeks to 1.5 month

JavaScript

JavaScript is a scripting or programming language that allows you to implement complex features on web pages.

The time needed to learn: from 2 months to 4 months

JS frameworks

JS frameworks are JavaScript code libraries that have pre-written code to use for routine programming features and tasks.

The time needed to learn: from 1 month to 2 months

Version control, also known as source control, is the practice of tracking and managing changes to software code. Version control systems are software tools that help software teams manage changes to source code over time.
